Principles

  • Technical identity — Fira Code monospace everywhere
  • Dark-first — Dark theme default, light theme supported
  • 4px grid — All spacing derives from 4px base unit
  • Accessible — WCAG AA contrast, 44px touch targets
  • Minimal motion — 50ms base, respects reduced-motion
